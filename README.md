# Sales Dashboard (Power BI)

##  Project Overview

This project is an interactive Sales Dashboard created using Power BI.
It helps analyze sales performance and provides insights for business decision-making.

---

##  Objective

To design a dashboard that tracks key business metrics such as sales trends and order performance.

---

##  Dataset

* Source: Kaggle Sales Dataset
* Contains fields like Date, Sales (Price Each), Product, and Order Number

---

##  Key Features

* KPI Cards showing Total Sales and Total Orders
* Line Chart for Sales over Time (trend analysis)
* Bar Chart for Sales by Product (comparison)
* Slicer (Filter) for interactive analysis

---

##  Insights

* Sales trends can be observed over time
* Top-performing products can be identified
* Users can filter data to explore specific products

---

##  Tools Used

* Power BI

---

##  File Included

* Sales_Dashboard.pbix

---

##  Outcome

This dashboard helps in understanding sales patterns and supports better business decisions through visual insights.
